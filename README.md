## 🛠️ Running Locally

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

> This will open the app at [http://localhost:3000](http://localhost:3000)
